To load in Visual Studio, open the ANTBridge.sln file.

To run the ANTBridge software, navigate into the bin/Release folder on the command-line and run the following command:

    ANTBridge.exe

To test the software, use the MulticastListener program:

    MulticastListener.exe

To change a configuration setting for either program, use the following command (with ANTBridge as an example):

    ANTBridge.exe [<setting> <value>]

To get a list of valid settings, run:

    ANTBridge.exe /?

For more information about the settings, see the ANTBridge Report document.
